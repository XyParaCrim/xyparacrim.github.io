# Redis位图

😫😫

Redis位图非常适合用于用户一年签到之类的场景

# 介绍

Redis位图以Bit数组的形式，若超出偏移范围会自动扩充。

# Cli

# setbit 和 getbit：单字符操作

通过位操作将字符串设置为Hello:

  • H：0b1101000
  • E：0b1100101
  • L：0b1101100
  • O：0b1101111
setbit s 1 1
setbit s 2 1
setbit s 4 1
setbit s 9 1
setbit s 10 1
setbit s 13 1
setbit s 15 1
get s
"he"

# bitcount和bitops

  • setbit 和 getbit：单字符操作
  • bitcount：用来统计指定范围内1的个数
  • bitops：查找指定范围内出现的一个1或者0
set w hello
# hello中1的个数
bitcount w
# h中1的个数
bitcount w 0 0
# 第一个0
bitops w 0

# bitfield

bitfield用于多位操作，拥有get、set和incrby三个子指令，指令提示：

bitfield key [GET type offset] [SET type offset value] [INCRBY type offset increment] [OVERFLOW WRAP|SAT|FAIL]
1

从h高4位无符号（u）是6

127.0.0.1:6379> bitfield w get u4 0 
(integer) 6
1
2

从l高4位有符号（i）数是-3

127.0.0.1:6379> bitfield w get i3 2
(integer) -3
1
2

批量执行

127.0.0.1:6379> bitfield w get u4 0 get u3 2 get i4 0 get i3 2
(integer) 6
(integer) 5
(integer) 6
(integer) -3
1
2
3
4
5

溢出策略

  • wrap：折返（255 + 1 = 0，127 + 1 = -128）
  • sat：饱和截断（不做操作）
  • fail：失败
127.0.0.1:6379> bitfield w overflow sat incrby u4 2 1
1

overflow与Java一样：64位
