# Redis-Info

通过info指令诊断Redis运行状态

# 主要诊断

  • Server：服务器运行的环境参数
  • Clients：客户端相关信息
  • Memory：服务器运行内存统计数据
  • Persistence：持久化信息
  • Stats：通用统计数据
  • Replication：主从复制相关信息
  • CPU：CPU使用信息
  • Cluster：集群信息
  • KeySpace：键值对统计信息

Redis每秒执行多少指令

> info stats
1

Redis连接多少客户端

> info clients
1

复制积压缓冲区多大

> info replication
1
