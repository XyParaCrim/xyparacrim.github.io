# Redis Cluster

  • Redis Cluster共分为三个对等的部分（无主从），每个部分可以由一个实例或一个集群组成。
  • 它们通过独特二进制协议通讯，并将所有数据分为16384个槽位。
  • key的定位通过客户端缓存其槽位信息，并在槽位不一致时，校正调整。

节点需要配置信息持久化到配置文件中，必须保证配置文件可写的权限
